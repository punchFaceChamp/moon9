## moon9::obj
- A dynamic/agnostic/serializable object library for POD data.
- MIT licensed.

## Features
- Create/Destroy objects at runtime
- Create/Delete/Modify object fields at runtime
- Object state serialization (*)
- Make object diffs (*)
- Patch diffs into objects (*)
